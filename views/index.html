The Timestamp Microservice

Welcome to the timestamp microservice! To access it enter this URL - https://tmp-fcc-three.glitch.me/api/timestamp/. Entering the URL will return a UNIX date as well as a UTC date in a JSON object.

If you'd like to retrieve UNIX and UTC timestamps for a specific date append the particular date of interest to the end of the above given URL. For example, entering the following: https://tmp-fcc-three.glitch.me/api/timestamp/1983-7-25 would return this JSON object:

{"unix":427939200000,"utc":"Mon, 25 Jul 1983 00:00:00 GMT"}

Multiple date formats may be appended to the end of the URL, including but not limited to the examples of 1983-7-25 and 42793900000. Go ahead and give it a try!
